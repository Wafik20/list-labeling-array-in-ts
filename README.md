# TypeScript Project

A basic TypeScript project setup.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Run with live reload (recommended for development):
```bash
npm run watch
```

4. Build the project:
```bash
npm run build
```

5. Run the built project:
```bash
npm start
```
